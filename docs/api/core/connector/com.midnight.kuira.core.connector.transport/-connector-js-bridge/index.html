ConnectorJsBridge

class ConnectorJsBridge(router: JsonRpcRouter, scope: CoroutineScope)

WebView JS Bridge transport for the DApp Connector.

Injected into a WebView as window.midnight. Web dApps call window.midnight.request(jsonRpcString) and get a JSON-RPC response.

Usage:

webView.addJavascriptInterface(jsBridge, "midnight")

From JavaScript:

const response = window.midnight.request(
  JSON.stringify({jsonrpc:"2.0", id:1, method:"getConnectionStatus", params:{}})
);
const result = JSON.parse(response);
