WalletThemes

object WalletThemes

The built-in appearance themes, in display order. Default (Kuira monochrome) is first and is the fallback for an unknown/absent persisted id.

Palette credits (all MIT): Catppuccin (catppuccin/catppuccin), Nord (nordtheme.com), Dracula (draculatheme.com), Tokyo Night (folke/tokyonight.nvim), Rosé Pine (rosepinetheme.com). Hex values taken verbatim from each project's published palette.
