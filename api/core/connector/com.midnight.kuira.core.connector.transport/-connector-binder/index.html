ConnectorBinder

class ConnectorBinder(val handler: ConnectedAPIHandler, router: JsonRpcRouter) : Binder

Android Binder transport for the DApp Connector.

Native Android apps bind to ConnectorService and get this Binder, which provides two access modes:

  1. handler — direct Kotlin API (type-safe, fastest, no serialization)

  2. handleJsonRpc — JSON-RPC string I/O (for apps that prefer the standard protocol)
